Story Mode README

目的

このディレクトリは、GUNDAM BATTLE SLOT のストーリーモード専用仕様を管理する。

ストーリーモードは通常対戦とは別に、以下の専用要素を持つ。

- クリエイトガンダム系列の育成
- クリエイトガンダムラボ
- スロット入れ替え
- 装備品管理
- エネルギー管理
- リロード武器管理
- 同行機体管理
- 同行機体育成
- ストーリー進行フラグ
- チャプター別戦闘定義
- 報酬管理

---

推測禁止

ストーリーモード実装時も、既存開発ルールに従う。

- 実コード未確認で既存処理を前提にしない
- 未確定の数値を仮置きしない
- 説明文依存で内部処理を作らない
- 仕様変更時はこの README または同階層の仕様ファイルを更新する
- 実装時は最新の "index.html" / "js" 実コードを確認する

初期特殊行動

特殊行動1. リロード

行動権を消費して、リロードが必要な武器を指定してリロードする。

リロードは一括ではなく、武器種指定で行う。

特殊行動2. エネルギーチャージ

行動権を消費して、エネルギーの充填を行う。

特殊行動3. 装備品1

装備品を使用する。

特殊行動4. 装備品2

装備品を使用する。

装備品は最大2個まで。

特殊行動5. エネルギー調整

エネルギー武器のエネルギーを同量追加消費して、ダメージ値を元の値の1.5倍へ増加させる。

加算累積可能。

クリエイトガンダム系列共通仕様

各スロットは、条件を満たすとクリエイトガンダムラボで入れ替え可能。

クリエイトガンダム系列は以下を持つ。

- HP
- 回避ストック
- エネルギー
- 使用可能コスト
- スロット構成
- 装備品
- クリエイトスキル

HP、回避ストック、エネルギー、武器性能、装備品、スロット登録は、使用可能コストの範囲内で構築する。

---

リロード武器仕様

リロード武器は、ダメージ値・攻撃回数とは別に弾数を管理する。

例：

汎用マシンガン
10ダメージ × 3回
射撃
リロード
弾数10
1ターン1装填

この場合、1回選択すると

10ダメージ × 3回を発動
弾数10 → 7

となる。

リロード属性表示

リロード属性は "[R]" で表記する。

ターン経過リロード

武器種ごとにターン経過時の装填値を持つ。

例：

1ターン1装填
2ターン1装填
5ターン1装填

特殊行動リロード

特殊行動「リロード」は、リロード対象の武器種を指定して行う。

---

リロード武器の追撃仕様

弾数が余っている場合、所持回避1消費で追撃可能。

回避が0の状態でも、1回だけ追撃可能。

回避0で追撃した場合、その追撃の会心率は倍になる。

---

エネルギー武器仕様

クリエイトガンダム系列はエネルギーを持つ。

プロトクリエイトガンダムの初期エネルギーは100。

エネルギー初期値は条件で増減する。

エネルギー回復

プロトクリエイトガンダムは、1ターンにエネルギーを5回復する。

エネルギー武器

エネルギー武器は、攻撃時に設定されたENを消費する。

エネルギーが0の場合は不発となる。

エネルギー調整

特殊行動「エネルギー調整」により、エネルギー武器の消費ENを増やしてダメージを増加できる。

この調整は事前設定型とする。

実際のスロット選択時には、調整後の数値を参照する。

---

同行機体仕様

同行機体もストーリーモード内で育成可能。

各同行機体は以下を持つ。

- 解放状態
- 同行可能状態
- レベル
- Exp
- ストーリー専用補正
- 装備または強化状態

同行機体は通常対戦とは別管理とする。

---

ストーリー限定レベル補正

ストーリーモードでは各機体ごとにレベルを持つ。

レベル上昇時の恩恵：

レベル値の半分の数値が会心率に蓄積
レベル値の半分の数値分、ダメージを軽減

例：

Lv2  → 会心率 +1 / ダメージ軽減1%
Lv10 → 会心率 +5 / ダメージ軽減5%
Lv50 → 会心率 +25 / ダメージ軽減25%
